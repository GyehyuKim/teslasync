# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

A research/discussion workspace, not a software project. The only content is
`research_brief.md` — a brief evaluating whether a Tesla USB port can stream
in-car dashcam footage to a phone over Bluetooth/wireless, plus existing
products (Tesla app Dashcam Viewer, teslausb, TesClip, Perception) and DIY
patterns (SBC emulating a USB drive, then archiving + serving a web UI).

There is no build, no tests, no dependencies. Work here is reading, research,
and editing Markdown.

## Conventions

- Deliverables are answered in **Korean**, concise bullets (see the brief's
  "Output format" section).
- The brief separates **verified findings** from **decisions needed** — keep
  that split when extending it; don't fold unverified claims into the findings list.
